# ➕➖✖️➗ Operadores e Expressões 

Operadores em JavaScript permitem manipular valores. Alguns exemplos incluem:

- **Aritméticos**: `+`, `-`, `*`, `/`, `%`
- **Atribuição**: `=`, `+=`, `-=`, `*=`
- **Comparação**: `==`, `===`, `!=`, `<`, `>`
- **Lógicos**: `&&`, `||`, `!`

Exemplo:

```javascript
let a = 10;
let b = 20;
console.log(a + b); // Soma: 30
console.log(a > b); // Comparação: false
```
