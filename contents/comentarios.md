# 📝 Comentários 

Comentários ajudam a explicar partes do código e são ignorados pelo interpretador. Use:

- **Linha única**: `//`.
- **Multilinha**: `/* */`.

Exemplos:

```javascript
// Isso é um comentário de linha única
```

```javascript
/*
Isso é um comentário
multilinha
*/
```

**Atalho para comentar:**
- Windows/Linux: `Ctrl + /`
- MacOS: `Cmd + /`
